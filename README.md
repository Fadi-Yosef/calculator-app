# Calculator Application

Console-based calculator built with Java and Maven.

## Features
- Addition, subtraction, multiplication, division
- Multi-operand support for + and -
- Advanced math (power, sqrt)
- JUnit 5 tests
- Optional Swing GUI

## How to Run

Run console app:
ConsoleCalculatorAPP.java

Run GUI:
CalculatorGUI.java

Run tests:
mvn test
